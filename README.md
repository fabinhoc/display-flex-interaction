## Posicionando elemento na tela com display flex

<p>use as teclas 
    <code>&larr;</code>
    <code>&rarr;</code>
    <code>&uarr;</code>
    <code>&darr;</code>
    para mover o quadrado na tela.
</p>

<p>use as teclas 
    <code>&crarr;</code> ou <code>space</code>
    para centralizar o quadrado.
</p>
